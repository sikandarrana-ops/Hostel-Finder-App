**HostelFinderApp – UX Evaluation \& Design Redesign**

This project focuses on evaluating and improving the usability of a hostel discovery platform using established UX research frameworks. The system was analyzed to identify usability issues affecting the user experience and accessibility across key user journeys, such as searching, viewing, and managing hostel listings.



The evaluation applied ISO 9241-11 usability standards and Nielsen’s 10 Usability Heuristics to systematically identify critical design flaws. Major issues included a lack of user guidance, CRUD functionality, and help documentation, which significantly impacted the overall user experience.
